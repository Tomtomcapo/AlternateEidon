# Concept du Mega-Prompt

## Résumé

Note méta sur la méthode de création d'Alternate Eidon — un projet développé en collaboration avec une IA générative.

## Contexte de création

Alternate Eidon est développé via une méthode de **"mega-prompt"** :
- Accumulation progressive de contexte
- Documents de spécification
- Fiches de personnages
- Règles de cohérence

## Le paradoxe créatif

L'IA utilisée pour écrire est elle-même une forme de "simulation" :
- Elle génère du texte selon des règles
- Elle n'a pas de conscience (probablement)
- Elle produit des personnages qui semblent conscients
- Miroir de Seth créant des Eidons

## Limites de la méthode

### Avantages
- Génération rapide
- Cohérence maintenue par le contexte
- Exploration de variations
- Documentation automatique

### Risques
- Contradictions si contexte incomplet
- Inventions non souhaitées
- Ton parfois générique
- Dépendance au prompt

Ce document sert de **rappel** pour l'IA :
- Ne pas inventer sans base canonique
- Vérifier la cohérence avant d'ajouter
- Poser des questions si ambiguïté
