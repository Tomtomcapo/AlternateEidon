# Niveaux d'Écriture

## Résumé

Ce document décrit les **couches narratives** d'Alternate Eidon — les différents niveaux de "réalité" et comment ils s'imbriquent.

## Les trois niveaux

### Niveau 1 : Le monde réel (Seth)
- Seth est un humain du monde réel
- Il a créé la simulation
- Il a copié ses proches comme Eidons
- **Il n'est plus là** (malade, mort, absent)

### Niveau 2 : La simulation (les Eidons)
- Le monde où vivent les personnages
- Les Eidons y sont quasi-divins
- Les habitants sont des consciences générées
- Le Primary est l'interface avec le niveau 1

### Niveau 3 : Le récit (l'auteur)
- L'auteur réel écrit Alternate Eidon
- Les personnages sont ses créations
- Le texte est le médium final

## Parallélisme intentionnel

La structure est **mise en abyme** :
- Seth crée un monde et y place des copies de ses proches
- L'auteur crée un monde et y place des personnages
- Les deux "créateurs" sont absents de leur création

Ce parallélisme est thématique, pas accidentel.

## Les Eidons comme métaphore

Les trois Eidons peuvent se lire comme des **réactions à la création** :

| Eidon | Réaction | Métaphore |
|-------|----------|-----------|
| [[characters/maelith|Maelith]] | Préserver, figer | L'œuvre qui refuse d'évoluer |
| [[characters/jadeus|Jadeus]] | Détruire puis reconstruire | L'œuvre qui se réinvente |
| [[characters/altert|Altert]] | Comprendre puis accepter | L'œuvre qui fait la paix avec son créateur |

## La question de l'agentivité

Qui a vraiment le contrôle ?

### Dans la simulation
- Seth a créé les règles
- Les Eidons agissent dans ces règles
- Mais Seth n'est plus là pour intervenir
- Les personnages sont-ils libres ?

### Dans le récit
- L'auteur définit l'histoire
- Les personnages "agissent" selon leur logique
- La cohérence interne devient contraignante
- L'auteur est-il vraiment libre ?

## Le Primary comme narrateur

Le Primary peut se lire comme :
- L'abstraction de Seth dans la simulation
- La "voix" de l'auteur dans le texte
- Le mécanisme qui maintient la cohérence
- Le gardien des règles

## Implications pour l'écriture

Cette structure impose des **contraintes narratives** :
- Les personnages doivent être cohérents
- Les règles du monde doivent être respectées
- Les contradictions brisent l'illusion
- La suspension d'incrédulité est fragile

## Détails canoniques

1. Trois niveaux : réel → simulation → récit
2. Parallélisme entre Seth/créateur et auteur/créateur
3. Les Eidons incarnent différentes réactions à la création
4. Le Primary est l'interface/narrateur
5. La cohérence interne est essentielle
