# Beyran — La Sentinelle d'Eldora

## Identité

**Beyran** est une ville frontière d'[[locations/eldora|Eldora]], face au [[locations/royaume-mort|Royaume Mort]] — pendant d'Eldora à [[locations/silentes|Silentes]].

### Attributs de base
- **Type** : Ville fortifiée / Garnison
- **Nation** : [[locations/eldora|Eldora]]
- **Position** : Frontière nord (face au Royaume Mort)
- **Fonction** : Défense, surveillance

## Position stratégique

Beyran est le **miroir de Silentes** — là où Eldora surveille son ennemi.

### Face au Royaume Mort
- Fortifications défensives
- Tours de guet
- Garnison permanente
- Préparation à l'invasion

### Différence avec Silentes
Beyran est **défensive**, pas offensive :
- Pas d'industrie lourde
- Pas de production d'armes
- Focus sur la résistance, pas la conquête
- Esprit des Principes d'Eldora

## Atmosphère

### Visuel
- Architecture classique fortifiée
- Murailles de pierre blanche
- Tours de guet sur la frontière
- Paysage de transition (vert d'Eldora / cramoisie du Royaume Mort)

### Ambiance
- Tension permanente
- Vigilance constante
- Solidarité entre habitants
- Fierté de garder la frontière

## Population

### Composition
- Soldats et gardes
- Familles de militaires
- Marchands de passage
- Réfugiés occasionnels

### Caractère
- Courage face à la menace
- Attachement aux Principes d'Eldora
- Méfiance envers le Nord
- Sacrifices quotidiens

## Le point de vue sur les Eidons

Beyran est l'un des rares endroits où on voit **concrètement** la menace de [[characters/jadeus|Jadeus]] :
- Les soldats connaissent les histoires
- Certains ont vu les escarmouches
- La peur est réelle, pas abstraite

## Détails canoniques

1. Ville frontière d'Eldora
2. Face à Silentes (Royaume Mort)
3. Position défensive, pas offensive
4. Point de tension permanent
5. Vue directe sur la menace
